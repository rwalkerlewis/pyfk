class pyfk.Config(model=None, source=None, receiver_distance=None, degrees=False, taper=0.3, filter=(0, 0), npt=256, dt=1.0, dk=0.3, smth=1.0, pmin=0.0, pmax=1.0, kmax=15.0, rdep=0.0, updn='all', samples_before_first_arrival=50, cuda=False)[source]¶

Bases: object

The configuration class used in generating Green’s function and the synthetic waveform.

Parameters

  • model (Optional[SeisModel]) – the Earth model used in calculation, defaults to None

  • source (Optional[SourceModel]) – the source model used in calculation, defaults to None

  • receiver_distance (Optional[Union[list, np.ndarray]]) – a list of receiver distance in km, defaults to None

  • degrees (bool, optional) – use degrees instead of km, defaults to False

  • taper (float, optional) – taper applies a low-pass cosine filter at fc=(1-taper)*f_Niquest, defaults to 0.3

  • filter (Tuple[float, float], optional) – apply a high-pass filter with a cosine transition zone between freq. f1 and f2 in Hz, defaults to (0, 0)

  • npt (int, optional) – the number of points, defaults to 256

  • dt (float, optional) – the sampling interval in seconds, defaults to 1.

  • dk (float, optional) – the non-dimensional sampling interval of wavenumber, defaults to 0.3

  • smth (float, optional) – makes the final sampling interval to be dt/smth, defaults to 1.

  • pmin (float, optional) – the min slownesses in term of 1/vs_at_the_source, defaults to 0.

  • pmax (float, optional) – the max slownesses in term of 1/vs_at_the_source, defaults to 1.

  • kmax (float, optional) – kmax at zero frequency in term of 1/hs, defaults to 15.

  • rdep (float, optional) – the depth for the receivers in km, defaults to 0.

  • updn (str, optional) – “up” for up-going wave only, “down” for down-going wave only, “all” for both “up” and “down”, defaults to “all”

  • samples_before_first_arrival (int, optional) – the number of points before the first arrival, defaults to 50

  • cuda (bool) – whether to use the cuda mode. if set PYFK_USE_CUDA=1, this flag will be ignored and will always use cuda, defaults to False

Raises

  • PyfkError – Must provide a list of receiver distance

  • PyfkError – Taper must be with (0,1)

  • PyfkError – Filter must be a tuple (f1,f2), f1 and f2 should be within [0,1]

  • PyfkError – npt should be positive.

  • PyfkError – dt should be positive.

  • PyfkError – dk should be within (0,0.5)

  • PyfkError – smth should be positive.

  • PyfkError – pmin should be within [0,1]

  • PyfkError – pmax should be within [0,1]

  • PyfkError – pmin should be smaller than pmax

  • PyfkError – kmax should be larger or equal to 10

  • PyfkError – the selection of phases should be either ‘up’, ‘down’ or ‘all’

  • PyfkError – samples_before_first_arrival should be positive

  • PyfkError – Must provide a source

  • PyfkError – Must provide a seisModel

class pyfk.SeisModel(model=None, flattening=False, use_kappa=False)[source]¶

Bases: object

SeisModel stores the Earth model

Parameters

  • model (np.ndarray) – numpy.ndarray model: a 2D numpy array storing the information of the 1D earth model, with the same format as FK. model has the following format (in units of km, km/s, g/cm3, each column): thickness vs vp_or_vp/vs [rho Qs Qp] rho=0.77 + 0.32*vp if not provided or the 4th column is larger than 20 (treated as Qs). Qs=500, Qp=2*Qs, if they are not specified. If the first layer thickness is zero, it represents the top elastic half-space. Otherwise, the top half-space is assumed to be vacuum and does not need to be specified. The last layer (i.e. the bottom half space) thickness should be always be zero. (if not, will use 0 anyway), defaults to None

  • flattening (bool, optional) – if the model and the source should be flatten, defaults to False

  • use_kappa (bool, optional) – if the third column of the model file is vp/vs ratio, defaults to False

Raises

  • PyfkError – Earth Model must be a 2D numpy array

  • PyfkError – Must provide at least three columns for the model

pyfk.calculate_sync(gf, config, az=0, source_time_function=None)[source]¶

Compute displacements in cm in the up, radial (outward), and transverse (clockwise) directions produced by different seismic sources

Parameters

  • gf (Union[List[Stream], Stream]) – the Green’s function list (or just an obspy Stream representing one distance) from calculate_gf; or read from the Green’s function database from FK

  • config (Config) – the configuration of sync calculation

  • az (Union[float, int], optional) – set station azimuth in degree measured from the North, defaults to 0

  • source_time_function (Optional[Trace], optional) – should be an obspy Trace with the data as the source time function, can use generate_source_time_function to generate a trapezoid shaped source time function, defaults to None

Raises

  • PyfkError – az must be a number

  • PyfkError – must provide a source time function

  • PyfkError – check input Green’s function

  • PyfkError – delta for the source time function and the Green’s function should be the same

Returns

a list of three components stream (ordered as z, r, t)
