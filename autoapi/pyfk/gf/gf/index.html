pyfk.gf.gf.calculate_gf(config: Optional[Config] = None) → Union[List[Stream], List[Stream]][source]¶

Compute displacements in cm in the up, radial (outward), and transverse (clockwise) directions produced by different seismic sources

Parameters

config (Optional[Config], optional) – the configuration of calculating the Green’s function, defaults to None

Returns

if npt==2 or 1, return a 2D list, and each row represents the static displacements; otherwise, return a list of Stream, each stream keeps the order of GF as in FK, and the order of streams is the same as the receiver_distance.

Return type

Union[List[Stream], List[Stream]]

Parameters

  • model (SeisModel) – the Earth model

  • config (Config) – the configuration of calculating GF

  • src_layer (int) – the layer where the source is located (source at the bottom)

  • rcv_layer (int) – the layer where the receiver is located (receiver at the top)

  • taper (float) – the taper value

  • pmin (float) – the min slowness

  • pmax (float) – the max slowness

  • dk (float) – sampling interval of wavenumber

  • nfft2 (int) – the half of simulation points

  • dw (float) – sampling interval of frequency

  • kc (float) – the max wave number

  • flip (bool) – if the source and the receiver has been fliped

  • filter_const (float) – the const value used in filtering

  • dynamic (bool) – if performing the dynamic simulation

  • wc1 (int) – the starting point of wave number integration

  • wc2 (int) – the end point beyond which the filtering will take effect

  • t0 (np.ndarray) – the first arrival time

  • wc (int) – the end point beyond which the taper will take effect

  • si (np.ndarray) – the source matrix

  • sigma (float) – the value to supress the numerical noise
