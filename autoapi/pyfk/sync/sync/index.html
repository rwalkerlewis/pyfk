pyfk.sync.sync.calculate_sync(gf, config, az=0, source_time_function=None)[source]#

Compute displacements in cm in the up, radial (outward), and transverse (clockwise) directions produced by different seismic sources

Parameters

  • gf (Union[List[Stream], Stream]) – the Green’s function list (or just an obspy Stream representing one distance) from calculate_gf; or read from the Green’s function database from FK

  • config (Config) – the configuration of sync calculation

  • az (Union[float, int], optional) – set station azimuth in degree measured from the North, defaults to 0

  • source_time_function (Optional[Trace], optional) – should be an obspy Trace with the data as the source time function, can use generate_source_time_function to generate a trapezoid shaped source time function, defaults to None

Raises

  • PyfkError – az must be a number

  • PyfkError – must provide a source time function

  • PyfkError – check input Green’s function

  • PyfkError – delta for the source time function and the Green’s function should be the same

Returns

a list of three components stream (ordered as z, r, t)

Return type

List[Stream]
